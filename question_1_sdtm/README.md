Question 1: SDTM DS Domain Creation using {sdtm.oak}

Objective

Create an SDTM Disposition (DS) domain dataset from raw clinical trial data using the {sdtm.oak} package, following CDISC SDTM Implementation Guide v3.4 standards.

Overview

This solution demonstrates the creation of a standardized SDTM DS (Disposition) domain from raw clinical trial data. The DS domain captures information about protocol milestones and subject disposition events during the study.
